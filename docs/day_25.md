# День 25. Финальная сборка: что ты реально умеешь

## Опора на материалы ВШЭ

Все пройденные материалы и задачи 01-7…01-10.

## Зачем этот день

Последний день — не новая теория, а честная проверка. Если не можешь нарисовать стек, память или регистры — тему надо повторить.

## Главная мысль

Готовность — это способность решить задачу, прочитать фрагмент, нарисовать модель и объяснить, почему код работает.

## Mock exam structure

```text
Part A: write code
  01-7, 01-8, 01-9, 01-10

Part B: read code
  cmp/test/jcc
  stack frame
  array addressing
  struct offsets

Part C: explain
  security
  x87/floating point
```

## Быстрый reverse-пример

```asm
mov eax, [ebp+8]
add eax, [ebp+12]
ret
```

Возможный C-смысл:

```cpp
return arg1 + arg2;
```

## Критерии готовности

| Skill | Готов, если можешь... |
|---|---|
| Bits | решить 01-7/8/9/10 без подсказок |
| Flags | объяснить `cmp`, `test`, `jcc` |
| Stack | нарисовать `call`, `ret`, frame |
| ABI | написать простую CDECL-функцию |
| Addressing | объяснить `[base+index*scale+offset]` |
| Reverse | восстановить простой C из asm |
| x87 | объяснить стековую модель и `%f`/`double` |

## Мини-экзамен на 90 минут

1. Написать формулу и NASM-идею для 01-7.
2. Написать 01-8 через `(a&c)|(b&~c)`.
3. Написать branchless abs.
4. Написать формулу Пандоры.
5. Объяснить `cmp eax, ebx; jl L`.
6. Нарисовать фрейм функции с двумя аргументами и двумя locals.
7. Объяснить `lea eax,[edx+4*ecx+8]`.
8. Восстановить C-смысл простого asm-фрагмента.
9. Объяснить, что делает canary.
10. Объяснить, почему `printf("%f")` ждёт `qword`.

## Типовая финальная проблема

Можно сдать 01-7…01-10 и всё равно плохо понимать стек. Поэтому перед экзаменом повторять нужно не только домашки, но и:

- `[ebp+8]`;
- `call/ret`;
- `cmp/test/jcc`;
- `mov` vs `lea`;
- structures offsets.

## Финальный принцип

> Если ты можешь объяснить тему другу на листочке с одной схемой — ты её понял.
