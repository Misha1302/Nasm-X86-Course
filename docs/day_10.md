## 01-7. Упаковка вектора

Нужно собрать 4 байта в 32-битное число:

```text
31..24      23..16      15..8       7..0
+-----------+-----------+-----------+-----------+
|     d     |     c     |     b     |     a     |
+-----------+-----------+-----------+-----------+
```

Формула:

```cpp
x = a | (b << 8) | (c << 16) | (d << 24);
```

NASM-идея:

```asm
mov eax, [a]
mov ebx, [b]
shl ebx, 8
or eax, ebx
mov ebx, [c]
shl ebx, 16
or eax, ebx
mov ebx, [d]
shl ebx, 24
or eax, ebx
```

Проверка: `1 2 3 4 -> 0x04030201`.

## 01-8. Masked merge

По каждому биту:

```text
if c[i] == 1 -> взять a[i]
if c[i] == 0 -> взять b[i]
```

Формула:

```cpp
d = (a & c) | (b & ~c);
```

Картинка:

```text
a      1 1 0 0
b      0 0 1 1
c      1 0 1 0
~c     0 1 0 1

a&c    1 0 0 0
b&~c   0 0 0 1
OR     1 0 0 1
```

## 01-9. Модуль числа без переходов

```cpp
mask = x >> 31;
answer = (x ^ mask) - mask;
```

NASM:

```asm
mov eax, [x]
mov edx, eax
sar edx, 31
xor eax, edx
sub eax, edx
```

Если `x >= 0`, `mask = 0`. Если `x < 0`, `mask = -1`, то есть все биты единицы.

## 01-10. Год на Пандоре

Месяцы: 41, 42, 41, 42, ...

```cpp
before = month - 1;
answer = before * 41 + before / 2 + day;
```

NASM-идея:

```asm
mov eax, [month]
sub eax, 1
mov ebx, eax
imul eax, 41
shr ebx, 1
add eax, ebx
add eax, [day]
```

## Типовые ошибки

- В 01-9 использовать `shr`, а не `sar`.
- В 01-8 перепутать `c` и `~c`.
- В 01-7 забыть сдвиг одного байта.
- Случайно использовать `jcc` там, где условные переходы запрещены.
