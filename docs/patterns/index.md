# Экзаменационные паттерны

Это не новая “теория сверху”. Это короткие карточки для задач, которые уже выходят за базовые 25 дней.

Формат каждой карточки:

```text
когда нужен -> идея -> shape -> типовые ошибки -> задачи
```

## Карта

| Если в задаче | Открыть |
|---|---|
| запрещены условные переходы | [Branchless-маски](./branchless) |
| `K` битов, popcount, значащие нули | [Битовые циклы](./bit_counting) |
| палиндромы, дроби, GCD | [Десятичные алгоритмы](./decimal) |
| рекурсия без массивов | [Рекурсия](./recursion) |
| строки, файлы, libc | [Строки и файлы](./strings_files) |
| условие требует 16-byte stack alignment | [libc и alignment](./libc_alignment) |
| перенос сегмента, linked list через массивы | [Массивная связность](./array_linked_list) |
| function pointer, varargs | [Advanced stack](./advanced_stack) |
| результат больше 64 бит | [Big integer](./bigint) |

## Как использовать

1. Сначала пройди базовые дни курса.
2. Потом найди паттерн по условию задачи.
3. Возьми только shape решения, не переписывай вслепую.
4. Проверь стек, регистры, формат ввода/вывода и крайние тесты.
