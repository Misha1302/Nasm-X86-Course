# Паттерн: десятичные алгоритмы

## Когда нужен

- развернуть число в десятичной записи;
- проверить палиндром;
- сложить дроби;
- сократить дробь через GCD.

## Разворот числа

C++-shape:

```cpp
rev = 0;
while (x != 0) {
    digit = x % 10;
    rev = rev * 10 + digit;
    x /= 10;
}
```

NASM-shape:

```asm
xor esi, esi        ; rev = 0
mov ecx, 10

.loop:
    test eax, eax
    je .end

    xor edx, edx
    div ecx         ; eax = x / 10, edx = x % 10

    imul esi, 10
    add esi, edx
    jmp .loop

.end:
```

## GCD через Евклида

```text
while b != 0:
    r = a % b
    a = b
    b = r
```

После цикла `a` — gcd.

## Сложение дробей

```text
num = num * y + x * den
den = den * y
g = gcd(num, den)
num /= g
den /= g
```

## Частые ошибки

| Ошибка | Почему плохо |
|---|---|
| забыть `xor edx, edx` перед `div 10` | делится `edx:eax`, а не только `eax` |
| портить `ecx=10` внутри цикла | следующий `div` сломается |
| не сократить дробь в конце | ответ не в требуемом виде |
| считать палиндром полученным раньше `N` шагов | в задаче может требоваться ровно после `N` шагов |
