# Паттерн: branchless-маски

## Когда нужен

Когда в условии написано:

```text
Запрещается использовать инструкции условной передачи данных и управления.
```

То есть нельзя `jcc` и `cmov`, но всё ещё можно считать формулы, маски, сдвиги и `and/or/xor`.

## За 30 секунд

- Маска для выбора должна быть `0` или `FFFFFFFFh`, не `0` или `1`.
- Выбор без `if`:

```text
answer = (a & mask) | (b & ~mask)
```

- `mask = FFFFFFFFh` выбирает `a`.
- `mask = 00000000h` выбирает `b`.

## Базовый шаблон

```asm
; eax = a
; edx = b
; ecx = mask: 0 или FFFFFFFFh

and eax, ecx
not ecx
and edx, ecx
or eax, edx
```

## Маска знака

```asm
mov edx, eax
sar edx, 31        ; edx = 0 или FFFFFFFFh
```

Так работает `abs(x)` без переходов.

## Ceil division

Для положительных чисел:

```text
ceil(a / b) = (a + b - 1) / b
```

Например, количество коробок:

```text
boxes = (items + D - 1) / D
```

## Частые ошибки

| Ошибка | Почему плохо |
|---|---|
| использовать маску `0/1` | выберется только младший бит |
| использовать `shr` вместо `sar` для знака | отрицательное число не даст `FFFFFFFFh` |
| случайно вставить `cmp/jcc` | нарушается запрет задачи |
| забыть, что `not` портит маску | сохрани копию, если маска ещё нужна |
