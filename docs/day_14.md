# День 14. `switch`, jump table и fall-through

## Опора на материалы ВШЭ

`Slides2026-10.pdf`, `Slides2026-0xB.pdf`: `switch`, jump table, Duff’s Device.

## Зачем этот день

На экзамене могут дать таблицу переходов и спросить: сколько `case`, где `default`, какие ветки объединены. Это reverse-задача, но довольно алгоритмическая.

## Главная мысль

Jump table — это массив адресов. Индекс выбирает, на какую метку прыгнуть.

```asm
cmp eax, 5
ja .default
jmp [.table + 4*eax]

.table:
dd .case0
dd .case1
dd .case2
dd .case3
dd .case4
dd .case5
```

Почему `4*eax`? В IA-32 адрес занимает 4 байта.

## Разбор примера

```asm
add eax, 2
cmp eax, 6
ja .L2
jmp [.L8 + 4*eax]

section .rodata
.L8 dd .L3, .L2, .L4, .L5, .L6, .L6, .L7
```

Как читать:

1. Сначала `eax = x + 2`, значит исходный `x` сдвинули, чтобы диапазон начинался с 0.
2. `cmp eax, 6; ja .L2` — если индекс больше 6, идём в default.
3. Таблица имеет 7 элементов: индексы `0..6`.
4. `.L6` встречается дважды — два `case` ведут в один код.
5. `.L2` может быть default или объединённая ветка.

## Таблица паттернов

| Pattern | Meaning |
|---|---|
| `cmp eax,max` + `ja default` | проверка выхода за диапазон |
| `jmp [table+4*eax]` | косвенный переход по таблице |
| same label twice | объединённые cases |
| no `jmp end` | fall-through |
| default label in table | missing/combined/default branch |

## Mini-check

В примере выше исходные значения `x` соответствуют индексам `eax = x+2`, значит таблица покрывает `x = -2..4`.

## Мини-челленджи

1. Какие два индекса ведут в `.L6`?
2. Какой диапазон исходных `x` покрывает таблица?
3. Зачем нужен `add eax, 2`?

1. Индексы 4 и 5.
2. `x = -2..4`.
3. Чтобы превратить отрицательный диапазон `case` в индексы `0..6`.
