# 01-4. Книжки

## Условие

К началу 2011 года в библиотеке было `X` книг. Каждый год:

1. в начале года покупают `N` книг;
2. к концу года теряют `M` книг.

Нужно узнать, сколько книг будет к началу года `Y`.

Запрещены условные передачи и условное управление.

## Ввод

```text
X N M Y
```

## Вывод

Одно число: количество книг к началу года `Y`.

## Ограничения

- `X, N, M <= 100000`;
- `2011 < Y <= 10000`;
- гарантируется, что книги не закончатся до начала года `Y`.

## Решение

Условия тут вообще не нужны. Надо только вывести формулу.

Сколько лет прошло:

```text
years = Y - 2011
```

Изменение за один год:

```text
delta = N - M
```

Ответ:

```text
answer = X + years * delta
```

## NASM-shape

```asm
mov eax, [Y]
sub eax, 2011        ; eax = years

mov ecx, [N]
sub ecx, [M]         ; ecx = delta

imul eax, ecx        ; eax = years * delta
add eax, [X]         ; eax = answer
```

## Ошибки

| Ошибка | Почему плохо |
|---|---|
| считать с 2012 года | нужно число лет `Y - 2011` |
| делать цикл по годам | условные переходы запрещены и цикл не нужен |
| читать через `%u`, потом использовать signed-логику | `N - M` может быть отрицательным |
