# 01-4. Книжки

## Коротко

Это задача на одну формулу. Условные переходы не нужны.

<details>
<summary>Идея решения</summary>

К началу года `Y` прошло:

```text
years = Y - 2011
```

Каждый год библиотека меняется на:

```text
delta = N - M
```

Ответ:

```text
answer = X + years * delta
```

</details>

## Где может сломаться

- перепутать `Y - 2011` и `Y - 2012`;
- забыть, что `N - M` может быть отрицательным;
- использовать условный переход, хотя он не нужен.
