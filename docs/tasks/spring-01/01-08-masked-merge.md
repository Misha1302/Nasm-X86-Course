# 01-8. Masked merge

## Условие

Даны три unsigned 32-bit числа `a`, `b`, `c` в hex.

Нужно вывести `d`, где каждый бит выбирается так:

```text
если c[i] = 1 -> d[i] = a[i]
если c[i] = 0 -> d[i] = b[i]
```

Запрещены условные передачи и условное управление.

## Ввод

```text
a b c
```

Числа записаны в шестнадцатеричном виде.

## Вывод

Одно 32-битное число `d` в шестнадцатеричном виде.

## Ограничения

- все числа — unsigned 32-bit;
- работать надо сразу со всеми 32 битами.

## Решение

Маска `c` показывает, где брать биты из `a`.

```text
a & c
```

Маска `~c` показывает, где брать биты из `b`.

```text
b & ~c
```

Обе части не пересекаются, значит их можно склеить через `or`:

```text
d = (a & c) | (b & ~c)
```

## NASM-shape

```asm
mov eax, [a]
and eax, [c]

mov ecx, [c]
not ecx
and ecx, [b]

or eax, ecx
```

Для ввода/вывода удобно:

```asm
fmtIn  db "%x %x %x", 0
fmtOut db "%x", 10, 0
```

## Ручные проверки

| `c` | Ответ |
|---|---|
| `0` | полностью `b` |
| `FFFFFFFF` | полностью `a` |
| `FF00FF00` | часть битов из `a`, часть из `b` |

## Ошибки

| Ошибка | Почему плохо |
|---|---|
| `(a & ~c) | (b & c)` | выбор будет наоборот |
| забыть `not c` для `b` | обе части будут брать одни и те же позиции |
| печатать через `%d` | hex-задача требует hex-вывод |
