# 01-8. Masked merge

## Коротко

Для каждого бита берём `a`, если в `c` стоит 1, иначе берём `b`.

<details>
<summary>Идея решения</summary>

Формула:

```text
d = (a & c) | (b & ~c)
```

`a & c` оставляет биты `a` там, где маска равна 1.

`b & ~c` оставляет биты `b` там, где маска равна 0.

</details>

## Где может сломаться

- перепутать `c` и `~c`;
- напечатать через `%d`, хотя удобнее hex/unsigned;
- думать, что нужен `if` для каждого бита.
