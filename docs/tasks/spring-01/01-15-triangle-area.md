# 01-15. Площадь треугольника

## Коротко

Плавающая точка не обязательна. Удобнее считать удвоенную площадь.

<details>
<summary>Идея решения</summary>

Удвоенная ориентированная площадь:

```text
det = (x2-x1)*(y3-y1) - (x3-x1)*(y2-y1)
```

Нужна площадь:

```text
absDet / 2
```

Для вывода с одним знаком:

```text
integerPart = absDet / 2
fractionDigit = (absDet % 2) * 5
```

Печатаем:

```text
integerPart.fractionDigit
```

</details>

## Где может сломаться

- забыть модуль `det`;
- использовать условный переход для abs;
- попытаться без нужды решать через x87;
- ошибиться с `.0` и `.5`.
