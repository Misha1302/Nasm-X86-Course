# 01-14. Огород

## Условие

Есть поле `N * M`. С каждого квадратного метра собрали `K` свёкол.

Свёклу разложили по коробкам по `D` штук. Последняя коробка может быть неполной.

Границу проходят 8 марта в `X` часов `Y` минут.

- ночью с `00:00` до `05:59` проходят все коробки;
- в другое время коробки нумеруют с нуля и задерживают номера, кратные 3.

Нужно вывести количество коробок, которые прошли границу.

Запрещены условные передачи, условное управление и явное использование `EFLAGS`.

## Ввод

```text
N M K D X Y
```

## Вывод

Одно число: количество прошедших коробок.

## Ограничения

- `1 <= N, M <= 1000`;
- `0 <= K <= 1000`;
- `1 <= D <= 10000`;
- `0 <= X <= 23`;
- `0 <= Y <= 59`.

## Решение

### 1. Считаем урожай

```text
items = N * M * K
```

### 2. Считаем коробки

Коробка может быть неполной, значит нужно деление вверх:

```text
boxes = ceil(items / D) = (items + D - 1) / D
```

Если `K = 0`, то `items = 0`, и формула даст `boxes = 0`.

### 3. Считаем дневной ответ

Днём задерживают коробки с номерами:

```text
0, 3, 6, 9, ...
```

Среди номеров `0..boxes-1` таких коробок:

```text
blocked = ceil(boxes / 3) = (boxes + 2) / 3
```

Значит:

```text
dayAnswer = boxes - blocked
```

### 4. Считаем ночной ответ

```text
nightAnswer = boxes
```

### 5. Выбираем без `if`

Ночь определяется только часом:

```text
night если X < 6
```

Так как `X` от 0 до 23, можно получить маску арифметикой:

```asm
mov ecx, [X]
sub ecx, 6
sar ecx, 31        ; ecx = -1 если X < 6, иначе 0
```

Дальше branchless-select:

```text
answer = (nightAnswer & mask) | (dayAnswer & ~mask)
```

## NASM-shape

```asm
; eax = items = N * M * K
mov eax, [N]
imul eax, [M]
imul eax, [K]

; boxes = (items + D - 1) / D
add eax, [D]
dec eax
xor edx, edx
div dword [D]
; eax = boxes
```

Для `blocked = (boxes + 2) / 3`:

```asm
mov eax, [boxes]
add eax, 2
xor edx, edx
mov ecx, 3
div ecx
; eax = blocked
```

## Ошибки

| Ошибка | Почему плохо |
|---|---|
| `boxes = items / D` | забыта неполная последняя коробка |
| считать задержанные как `boxes / 3` | коробка `0` тоже кратна трём |
| использовать `cmp/jcc` для ночи | условное управление запрещено |
| сделать маску `0/1` | для выбора всего числа нужна `0/FFFFFFFFh` |
| использовать `Y` для ночи | весь час `0..5` ночной, минуты не меняют решение |
