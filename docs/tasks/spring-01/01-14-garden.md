# 01-14. Огород

## Коротко

Нужно посчитать коробки и без `if` выбрать ночной или дневной ответ.

<details>
<summary>Идея решения</summary>

Сначала урожай:

```text
items = N * M * K
boxes = ceil(items / D)
```

Ночью проходят все коробки:

```text
nightAnswer = boxes
```

Днём задерживаются номера `0, 3, 6, ...`.

Количество задержанных среди `0..boxes-1`:

```text
blocked = ceil(boxes / 3)
dayAnswer = boxes - blocked
```

Потом нужен branchless-выбор по `X < 6`.

</details>

## Где может сломаться

- считать `boxes = items / D` без округления вверх;
- забыть, что коробки нумеруются с нуля;
- получить маску `0/1` вместо `0/FFFFFFFFh`;
- случай `K = 0`: коробок ноль.
