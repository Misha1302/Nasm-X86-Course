# 02-12. Двоичные нули лайт

## Коротко

`N <= 100000`, поэтому можно перебрать все числа от `1` до `N`.

<details>
<summary>Идея решения</summary>

Для каждого `x`:

```text
len = bit_length(x)
ones = popcount(x)
zeros = len - ones
```

Если `zeros == K`, увеличиваем ответ.

</details>

## Где может сломаться

- считать ведущие нули значащими;
- забыть, что диапазон начинается с `1`;
- не обработать `K=0`;
- испортить исходное `x`, если оно ещё нужно.
