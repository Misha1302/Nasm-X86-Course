# 02-14. Двоичные нули

## Коротко

`N` до `10^9`, перебор не проходит. Нужен подсчёт по битам.

<details>
<summary>Идея решения</summary>

Считаем количество чисел `<= N`, у которых ровно `K` значащих нулей.

1. Для всех длин меньше `bit_length(N)` добавляем:

```text
C(len - 1, K)
```

потому что старший бит всегда `1`, а остальные `len-1` позиций могут содержать нули.

2. Для длины как у `N` идём по битам сверху вниз. Когда в `N` стоит `1`, можно поставить в этом месте `0` и заполнить остаток любыми битами с нужным числом нулей.

</details>

## Где может сломаться

- пытаться перебрать до `10^9`;
- считать ведущие нули;
- ошибиться с `C(n,k)`, когда `k < 0` или `k > n`;
- забыть учесть само число `N`.
