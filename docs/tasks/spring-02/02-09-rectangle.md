# 02-9. Прямоугольник

## Коротко

Вершины идут в произвольном порядке. Сначала находим `minX/maxX/minY/maxY`.

<details>
<summary>Идея решения</summary>

После чтения 4 вершин:

```text
minX = min(all x)
maxX = max(all x)
minY = min(all y)
maxY = max(all y)
```

Точка строго внутри:

```text
minX < px < maxX
minY < py < maxY
```

Если хотя бы одно условие не выполнено — `NO`.

</details>

## Где может сломаться

- принять границу за внутреннюю точку;
- не учесть произвольный порядок вершин;
- использовать unsigned-сравнение для координат;
- забыть вывести строку, а не число.
