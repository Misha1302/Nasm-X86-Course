# 02-3. Локальные экстремумы

## Коротко

Читаем массив, проверяем элементы `1..N-2`, отдельно сохраняем индексы минимумов и максимумов.

<details>
<summary>Идея решения</summary>

Для каждого `i`:

```text
min: a[i] < a[i-1] && a[i] < a[i+1]
max: a[i] > a[i-1] && a[i] > a[i+1]
```

Первый и последний элементы не проверяются.

Индексы удобно складывать в два массива:

```text
mins[]
maxs[]
```

Сначала выводим количество и индексы минимумов, потом количество и индексы максимумов.

</details>

## Где может сломаться

- проверить `i=0` или `i=N-1`;
- использовать unsigned-сравнение для signed чисел;
- вывести индексы не с нуля;
- забыть случай `N < 3`.
