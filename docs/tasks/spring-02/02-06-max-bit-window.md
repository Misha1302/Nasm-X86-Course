# 02-6. Поиск максимальной последовательности бит

## Коротко

Скользящее окно из `K` битов по всем позициям числа.

<details>
<summary>Идея решения</summary>

```text
mask = (1 << K) - 1
answer = 0

for shift = 0..32-K:
    candidate = (N >> shift) & mask
    answer = max(answer, candidate)
```

Сравнение кандидатов — unsigned.

</details>

## Где может сломаться

- забыть, что переменный сдвиг идёт через `cl`;
- сделать неверную маску для `K=31`;
- использовать signed max;
- пройти не все позиции.
