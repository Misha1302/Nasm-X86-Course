# Краткие разборы задач

Это не банк готового NASM-кода. Формат другой: быстро понять условие, вход/выход, ограничения, алгоритм и места, где обычно ломается решение.

## Как читать разбор

На каждой странице есть один и тот же каркас:

| Блок | Зачем |
|---|---|
| Условие | что надо получить |
| Ввод / вывод | что именно читает и печатает программа |
| Ограничения | какие размеры важны для алгоритма и типов |
| Решение | идея, псевдокод, NASM-shape |
| Ошибки | типовые причины WA/RE |
| Где в курсе | какие главы/паттерны открыть |
