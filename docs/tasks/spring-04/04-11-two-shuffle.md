# 04-11. 2-перемешивание

## Коротко

Нельзя физически двигать массив на каждом запросе. Нужен linked list через массивы `next/prev`.

<details>
<summary>Идея решения</summary>

Изначально:

```text
1 -> 2 -> 3 -> ... -> N
```

Храним:

```text
next[value]
prev[value]
head
```

Операция `[b..c]` в начало:

```text
before = prev[b]
after  = next[c]
отрезать сегмент
приклеить его перед текущим head
head = b
```

В конце выводим проходом от `head` по `next`.

</details>

## Где может сломаться

- делать O(N) перенос на каждый запрос;
- не обработать случай, когда `b` уже `head`;
- забыть обновить `prev[head]`;
- сломать связи после `c`.
