# 03-18. Знаковое произведение

## Коротко

Это advanced-задача: произведение трёх 32-bit чисел может не поместиться в 64 бита.

<details>
<summary>Идея решения</summary>

Нужна многоразрядная арифметика.

План:

1. определить знак результата;
2. перейти к абсолютным значениям множителей;
3. посчитать произведение в нескольких `dword`;
4. вывести `-`, если результат отрицательный;
5. вывести decimal через повторное деление большого числа на 10.

</details>

## Где может сломаться

- оставить только `eax` или `edx:eax`;
- потерять знак;
- не уметь печатать 96-bit число;
- забыть случай нуля.
