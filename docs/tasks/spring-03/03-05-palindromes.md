# 03-5. Палиндромы

## Коротко

Нужна функция разворота числа в десятичной записи.

<details>
<summary>Идея решения</summary>

Функция:

```text
reverse_decimal(x)
```

Цикл на `N` шагов:

```text
repeat N times:
    M = M + reverse_decimal(M)
```

После ровно `N` шагов проверяем:

```text
M == reverse_decimal(M)
```

Если да — `Yes` и число, иначе `No`.

</details>

## Где может сломаться

- остановиться раньше `N` шагов;
- забыть `xor edx, edx` перед `div 10`;
- использовать signed division для unsigned числа;
- испортить число при проверке палиндрома.
