# 03-9. Недостаточные числа

## Коротко

Нужна отдельная CDECL-функция `is_deficient(n)`.

<details>
<summary>Идея решения</summary>

Для числа `n` считаем сумму собственных делителей:

```text
sum = 0
for d = 1..n-1:
    if n % d == 0:
        sum += d
```

Если:

```text
sum < n
```

число недостаточное.

Основной цикл перебирает натуральные числа и считает, сколько недостаточных уже найдено, пока не дойдёт до `K`.

</details>

## Где может сломаться

- не вынести проверку в функцию;
- забыть CDECL cleanup;
- перепутать остаток `edx` после `div/idiv`;
- использовать слишком медленный верхний предел без необходимости.
